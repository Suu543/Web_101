Animation - Transition

  • 프로퍼티 값 변경을 기준 - 자동으로 적용되지 않고 특정 프로퍼티 값이 변경되는 이벤트가 발생시 적용됨
  • transition-property: all(기본값) 어떤 프로퍼티에 반응 할 것인가
  • transition-duration: 0s(끼본값) 지속시간
  • transition-timing-function: ease(기본값) 어떤 비율로 변화를 적용할 것인가
  • transition-delay: 0s(기본값) 얼마의 지연후 적용할 것인가
  • transition: property duration function delay - 축약형


















Animation - Animation

  • 프로퍼티 값 변경 여부에 상관없이 바로 적용되는 이벤트 더 부드러운 느낌을 느낄 수 있다.
  • @keyframes 애니메이션이름 {} - from to or 0% ~ 100%
  • animation-name: 정의한 keyframes 이름
  • animation-duration: 정의한 애니메이션 지속 시간 - 처음 ~ 끝
  • animation-delay: 애니메이션 적용 지연 시간
  • animation-iteration-count: infinite: 정의한 애니메이션 반복 횟수
  • animation-direction: 애니메이션 진행 방향 from --> to or to --> from
  • animation-fill-mode: 애니메이션 실행 전 실행 후 상태 결정
  • animation-play-state: 애니메이션을 특정 이벤트에의해 도중에 멈추는데 이용
  • animation: name duration function delay iteration direction fill-mode play-state - 축약형


















Animation - Transform

  • transition은 특정 이벤트에 반응해 적용
  • animation은 자동으로 적용
  • transform은 transition에 가깝지만 이동, 회전, 확대축소, 비틀기 등의 기능을 지원 - 단, 다른 요소의 layout에 어떠한 영향도 미치지 않음
  • transform: translate(x, y) - 요소의 위치를 x축으로 x 만큼, y축으로 y 만큼
  • transform: scale(x, y) - 요소의 크기를 x축으로 x 만큼, y축으로 y 만큼
  • transform: skew(30deg) - 요소를 30도 만큼 비틀기, x & y 축으로
  • transform: rotate(180deg) - 요소를 180도 회전하기
